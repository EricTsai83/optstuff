## 2026-02-01

### Initial Release

- **Image Optimization API** — On-the-fly resize, format conversion (WebP, AVIF), and quality adjustment
- **Signed URL Security** — HMAC-SHA256 signatures for every request
- **Multi-Tenant Architecture** — Teams, projects, and API keys hierarchy
- **Domain Whitelisting** — Project-level referer domains and API key-level source domains
- **Encrypted Key Storage** — AES-256-GCM encryption for secret keys at rest
- **Rate Limiting** — Configurable per-minute and per-day limits per API key
- **Usage Analytics** — Request logs, bandwidth tracking, and top images
- **Dashboard** — Full project management, API key lifecycle, and developer tools
