## v1.0.0

Initial public release of OptStuff.

### Features

- On-the-fly image optimization via signed URL API
- HMAC-SHA256 URL signing with optional expiration
- Multi-tenant architecture (Teams → Projects → API Keys)
- AES-256-GCM encrypted API key storage
- Two-layer domain whitelisting (source domains + referer domains)
- Per-key rate limiting (sliding window, per-minute + per-day)
- Image operations: resize (`w`, `h`, `s`), format conversion (`f`), quality (`q`), fit modes, embed
- Supported formats: JPEG, PNG, WebP, AVIF (input); JPEG, PNG, WebP, AVIF (output)
- Dashboard with team/project/key management, usage analytics, and request logs
- `HEAD` fast-path for lightweight availability probes
- CDN-optimized cache headers (`Cache-Control: public, s-maxage=31536000, immutable`)
